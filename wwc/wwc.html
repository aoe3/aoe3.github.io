This is a statistical breakdown of the taolu medals counts for the World Wushu Championships. To be part of the dataset, a country must have placed first, second, or third in at least one taolu event from 1991 (1st WWC) to 2017 (14th WWC). Almost all stats were obtained and sorted from the IWUF website.

Sanda is not included in this dataset.

Please see the bottom of the page for info on the data sets.

Note 1: For the 4th WWC in the IWUF website, the file did not contain country affiliations for the taolu medalists, as seen here. As a result, the data will be pulled from this site. Special thanks to beijingwushuteam.com for having these records up!

Note 2: In the IWUF results, there are many occurrences where countries tie for first or second, and the next rank is 4th. There are also a few occurrences where countries tie for third. In all of these cases, a country's rank is recorded exactly as the IWUF reported it.

(hypothetical example 1: {place:2 ; nations:["JPN", "SGP"]} - Both Japan and Singapore will be represented as second place in the data. There will be no third place.)

(hypothetical example 2: {place:3 ; nations:["USA", "CAN"]} - Both the United States and Canada will be represented as third place in the data. There will still be a first and second place.)

Therefore, the number of third, second and first place winners will not be equal, unlike what you would expect if only one country could place in a certain spot.
