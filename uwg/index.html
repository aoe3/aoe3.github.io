This is a statistical breakdown of the Adult division of competition at University Wushu Games. Senior, Child, Youth, and Group A/B/C divisions are not displayed, as I felt a bit lazy after scraping data from TerpWushu's site (Sorry.) :p

No competitor is represented more than once in the data, and each competitor object created in the code contains an array of their participated events, so as to prevent duplicates from occurring.
